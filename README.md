# portifolio_project

## Overview
This project is a simple **personal portfolio website** created using only **HTML**.  
It is designed to introduce myself, showcase my skills, and provide a way for visitors to contact me.  
The project demonstrates the use of basic HTML elements such as headings, paragraphs, lists, tables, and forms.

##  Features
- **Homepage Introduction**: Welcoming text that explains the purpose of the portfolio.
- **About Section**: A detailed description of my background, hobbies,skills and career goals.
- **Contact Section**: An interactive form with Text input,Number input ,Radio buttons,Checkboxes,Range input,Submit and reset buttons. Labels are properly linked with `for` and `id` attributes for accessibility.

## Technologies Used
- **HTML5** only (no CSS or JavaScript in this version).
- Semantic HTML elements for structure and readability.

## How to Run
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Explore the sections and try out the contact form.
